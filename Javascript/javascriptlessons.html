Inline Javascript

Code: 

 <button onclick="alert('Thanks for Clicking. Click OK to close me.')">Click Me!</button>

Inline javascript is like inline css in many ways:

  1. limited functionality
  2. clutters code
  3. difficult to update, and
  4. lacks re-usability.

Internal Javascript

The alert when the page loads is an example of internal javascript. While it is possible to put the javascript anywhere on the page the convention is to place it in the head or right before the closing body tag.

Code: 

<script type="text/javascript">
	//This alert pops up when page has loaded
	alert("Page loaded!");
</script>

Arrays

Arrays are similar to variables in that they hold values but arrays are capable of holding lots of values at one time.

One way to define an array is to use var and new Array() like this:

var myArray = new Array();

Then you can add individual values to the array like this:

myArray[0] = "value1";
myArray[1] = "value2";
etc...

Note that the numbering of an array starts with 0 not 1.

There are other ways to define arrays:

my2ndArray=["rain","sun"]; //creates an array containing rain & sun

You can add elements to an array using:

my2ndArray.push("snow"); //adds snow at the end of the array

We can remove an item or items from an array or insert an item in a specific position using splice.

my2ndArray.push("snow"); //adds snow a 2nd time

my2ndArray.splice(3,1); //starts at 3 position and removes 1 element

my2ndArray.splice(1,0,"hail"); //adds hail between rain and sun

Adding .length after the array name in a console.log will tell you the number of elements in an array.

console.log(my3rdArray.length);

If Statements

If statements allow for alternate actions dependent on the truth of a condition. If it is raining -> wear a raincoat Using else if and else in conjunction with the if allow for greater options... If it is raining -> wear a raincoat, else if it is snowing -> wear a snowsuit, else just go out and play

Please enter a number between 1 and 10.

Code: 

	//compares user answer to requested number range and gives appropriate response
	var ifExample = document.getElementById("ifExample");
	var ifResponse = document.getElementById("ifResponse");
	var ifSubmit = document.getElementById("ifSubmit");
	var newResponse = "";

	ifSubmit.onclick = function ifTest() {
		newResponse = ifResponse.value;
		if (newResponse >= 1 && newResponse <= 10) {
			ifExample.innerHTML = "Please enter a number between 1 and 10.";
			alert("Thanks for cooperating!");
		} else if (newResponse < 1) {
			ifExample.innerHTML = newResponse + " is less than 1.  Please enter a number between 1 and 10.";
		} else if (newResponse > 10) {
			ifExample.innerHTML = newResponse + " is greater than 10.  Please enter a number between 1 and 10.";
		} else {
			ifExample.innerHTML = newResponse + " is not a number.  Please enter a number between 1 and 10.";
		}
		ifResponse.value='';
	}

For Loops

For loops allow you to repeat a process for a specified number of times or until a limit value is reached. For example, if I wanted to iterate through an array and save it's contents to a string to be printed on the screen instead of having to look at the console to check the array's values.

Note you can use HTML formatting in your string to make the output look any way you want. I choose to have my array contents output as an ordered list that started at 0 to reflect the numbering of the array itself.

Code: 

	//establishes an array of elements, loops through the array storing
	//the values in a formatted string, adds the string to the arrayContents
	//div for the user to see, then offers reset which resets colorList
	//and empties arrayContents div
	var favColors = ["blue","silver","pink","brown","grey"];
	var colorList = "<ol start=\"0\">";

	document.getElementById("revealContents").onclick = function showArray() {
		document.getElementById("revealContents").style.display="none";
		for ( var i=0; i<favColors.length; i++) {
			colorList = colorList + "<li>" + favColors[i] + "</li>";
		}
		colorList = colorList + "</ol>";
		document.getElementById("arrayContents").innerHTML = colorList;
		document.getElementById("hideContents").style.display="inline";
	}

	document.getElementById("hideContents").onclick = function hideArray() {
		document.getElementById("hideContents").style.display="none";
		document.getElementById("arrayContents").innerHTML = "";
		colorList = "<ol start=\"0\">";
		document.getElementById("revealContents").style.display="inline";
	}

While Loops

Like for loops, While Loops allow you to iterate through a process. The cool thing is they work when you don't know how many times you need to perform the loop. If the condition of your while loop must be true then the loop will stop when the condition becomes false. Likewise, if the condition must be false, the loop will stop when the condition becomes true.

See my favorite colors.

 

Code: 

	//uses a while loop to output contents of an array or clear
	//output on button click
	var view = document.getElementById("whileView");
	var clear = document.getElementById("whileClear");
	var i = 0;
	var colorString = "";
	var output = document.getElementById("whileOutput");

	view.onclick = function viewColors() {
		view.style.display = "none";
		while ( i < favColors.length) {
			if (colorString == "") {
				colorString = favColors[i];
			} else {
				colorString = colorString + ", " + favColors[i];
			}
			i++;
		}
		output.innerHTML = colorString;
		clear.style.display = "inline";
	}

	clear.onclick = function clearColors() {
		i = 0;
		colorString = "";
		output.innerHTML = " ";
		clear.style.display = "none";
		view.style.display = "inline";
	}

Functions

Functions are a way of breaking code into more manageable blocks that are easier to read and reusable. You define a function by using the word function followed by a name, typically using camel case, like alertMe and then parentheses (). The code that makes up the function is contained inside curly braces {} after the parentheses. Functions can have variables inside their parentheses, you can then pass a values for each variable in the function when you call it.

External

Like CSS, it doesn't take long for javascript to overwhelm an HTML file. Also like CSS, the solution to the problem is creating an separate, or external, file to hold the javascript.

To make your external javascript file you simply copy all the code between your script tags, name it, and add .js to the end of the name. Then add a src="yourfile.js" to your opening script tag and your page will run just like it did when the code was in the page but it will be a lot easier to read.
