jQuery

jQuery is a javascript library that sits on top of javascript and allows you to access certain functions more easily. It also qives you a bit more power allowing you to create complex interactive web apps more easily.

There are a wide assortment of plugins that expand the functionality of jQuery. One plugin most smartphone and tablet users will recognize is the pull to refresh plugin that refreshes the page when the uses swipes down on the top of the page. Another useful plugin is jQuery UI (user interface) which assists in creating dragable, dropable, and sortable elements to web apps.

Embedding jQuery

You can go to jQuery.com and download one of the available versions of jquery which will allow you to control which version of jquery you are coding against and to work offline. You can also link to jQuery.com or google's online code library which will always be the most current version and will tend to load faster than from your own site.

Regardless of your jQuery source you include it in a page by using script tags just like you did for javascript.

<script type="text/javascript" src="Scripts/jquery-1.11.2.min.js"></script>

Detecting A Click

Javascript and jQuery differ a little in syntax. Instead of calling document.getElementById("idName") for an element with an id you would use $("#idName"). You can also call elements by class $(".className"), or by element type $("div"). Additionally, you can name an instance of an element and then call the element by it type and name $("div[name='divName']"), which allows for identically styled divs to receive different actions/instructions.

Code:  
	$("div[name='square1']").click (function() {
		alert ("Left square clicked!");
	})

	var click = false;
	$("#circle").click(function(){
		if (click == false) {
			$("#circle").css("border","2px solid red").css("background","transparent").css("width","196px").css(
				"height","196px").css("border-radius","98px");
			click = true;
		} else {
			$("#circle").css("border","none").css("background","red").css("width","200px").css("height","200px"
				).css("border-radius","100px");
			click = false;
		}
	})

	var clicked = false;
	$("div [name='square2']").click (function () {
		if ( clicked == false ) {
			$("div [name='square2']").css("background","green");
			clicked = true;
		} else {
			$("div [name='square2']").css("background","Chartreuse");
			clicked = false;
		}
	})

Changing Website Content

We can use jQuery to change content similar to the way we change content with plain javascript. Click the circle below to see it in action.

Code:  
	var change = false;
	$("#changeP").click (function () {
		if (change == false) {
			$("#changeMe").html("The text has been changed! Click again to change it back.");
			change = true;
		} else {
			$("#changeMe").html("We can use jQuery to change content similar to the way we change content with plain
				javascript. Click the circle below to see it in action.");
			change = false;
		}
	})

You can also retrieve contents using the .html function in jQuery.

Code:  
	$("#retrieveP").click (function () {
		alert($("#retrieveMe").html());
	})

The above examples demonstrate a fairly consistent trait of jQuery. When function brackets are empty they return the value of the attribute. When they contain data they update the value of the attribute.

.hover allows you to update or return a value on hover instead of on click. Note that the original state does not automatically revert when you move the pointer out of hover range.

By using the calls to specific mouse events, like mouseenter, mouseleave, mouseover, mouseout, etc. you can achieve different effects.

Code:  
	var site = $("iframe").attr("src");
	$("#updateIframe").mouseenter (function () {
		if ( site == "blue.html" ) {
			$("iframe").attr("src","red.html");
		} else {
			$("iframe").attr("src","blue.html");
		}
		site = $("iframe").attr("src");
	})

Animating Content

JQuery handles animations very well and fading is just one of the animations built into it. Other jQuery animations include slide, toggle, and the ability to construct custom styling for animation.

The circle below will grow or shrink on click, depending on it's current size.

Code:  
	$("#growingCircle").click( function () {
		if ($(this).css("width") == "50px") {
			$(this).animate({
				width:"200px",
				height:"200px",
				borderRadius:"100px",
				margin:"40px"
			},500);
		} else {
			$(this).animate({
				width:"50px",
				height:"50px",
				borderRadius:"25px",
				margin:"10px"
			},500);
		}
	});

The circle below will grow, hold for 2 seconds, then shrink back to it's starting size.

Code:  
	$("#growCircle").click(function () {
		$(this).animate ( {
			width:"200px",
			height:"200px",
			borderRadius:"100px",
			margin:"40px"
		}, 500, function () {
			$(this).delay (2000).animate ( {
				width:"50px",
				height:"50px",
				borderRadius:"25px",
				margin:"10px"
			}, 500);
		});
	});

Loading Data with AJAX

AJAX stands for Asynchronous Javascript And XML. AJAX is a powerful javascript tool that works particularly well with jQuery. Asynchronous means that it runs in the background and other code can run in the foreground at the same time. AJAX won't run on the file protocol it needs to be run online.

The get() function will allow you to access other files but if there is a problem (like no internet) there is no way to let the user know. The ajax() function is more flexible and will let you define an action to be taken if the retrieval fails.

Regular Expressions

Regular expressions are not exclusive to jQuery or javascript, they are available in all programming languages. A regular expression is a pattern of characters that you can use to determine if a string is of a desired format (like phone number or email address).

To define a regular expression we use var regex = /term/; stringBeingChecked.match(regex); If the defined term is found in the string it returns the value of the regex/term, otherwise it returns null. If you want to make use of the return value you should store it in a variable: var result = stringBeingChecked.match(regex);

There are four flags that can be added after your regex term that expand it's functionality. i allows for a case-insensitive search, g searches for every match in the string, m is for a multi-line search, and y is a "sticky" search that starts at the current point in the target string. These flags can be use separately or together.

The paragraph below shows the results of an ig flagged search for /that/ on the paragraph above.

The word /that/ is found 3 times in the searched paragraph.

Code:  
	var regex = /that/ig;
	$("#regexResult").html($("#regexSearch").html().match(regex));

While it may not seem very useful in the previous example it actually allows for many useful comparisons and validations. You can ensure that a user enters an email address, phone number, or a password with the required capital, number, and special character.
