Bootstrap

Bootstrap is an opensource, completely free web framework from Twitter that allows you to create good-looking, responsive websites. Because users access the web from a variety of devices a page needs to look good on the small screen of a phone and on a tv sized screen. Responsive sites change layout and element size as screen size changes.

Like jQuery you can download a specific version of bootstrap and include it with your site files or you can use a hosted version that is always the most recent, stable version available.

The Grid System

Grid systems are used for creating page layouts through a series of rows and columns that house your content. Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. It includes predefined classes for easy layout options, as well as powerful mixins for generating more semantic layouts.

Breakpoints, widths at which the layout changes, rearrange the grid elements so that viewing is optimized at every viewing size

Bootstrap pages are set out as a series of rows and columns with nearly limitless combinations. Bootstrap comes with it's own css that allows it to respond appropriately to varying breakpoints.

Navbars

Bootstrap comes with a lot of built in functionality. One of these is the navbar classes that allow you to quickly and simply build fully responsive navigation on your site.

While some of the code involved in navbars and other built in functionality can be complex there are many good starter examples avialable at getbootstrap.com in the CSS, Components, and JavaScript sections.
