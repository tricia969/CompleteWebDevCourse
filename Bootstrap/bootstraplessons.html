Bootstrap

Bootstrap is an opensource, completely free web framework from Twitter that allows you to create good-looking, responsive websites. Because users access the web from a variety of devices a page needs to look good on the small screen of a phone and on a tv sized screen. Responsive sites change layout and element size as screen size changes.

Like jQuery you can download a specific version of bootstrap and include it with your site files or you can use a hosted version that is always the most recent, stable version available.

The Grid System

Grid systems are used for creating page layouts through a series of rows and columns that house your content. Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. It includes predefined classes for easy layout options, as well as powerful mixins for generating more semantic layouts.

Breakpoints, widths at which the layout changes, rearrange the grid elements so that viewing is optimized at every viewing size

Bootstrap pages are set out as a series of rows and columns with nearly limitless combinations. Bootstrap comes with it's own css that allows it to respond appropriately to varying breakpoints.

Bootstrap Components

Bootstrap comes with a lot of built in functionality. The Components section of the website provides instructions and examples for them.

Glyphicons

Over 250 glyphs from the Glyphicons Haflings set are available free of charge through Bootstrap. Glyphicons cannot be mixed with other classes, but they are easy enough to use anywhere simply by enclosing them in a span. You can also apply custom classes to the glyphs to change their appearance the same way you would any other font.

Glyphs are great for highlighting sections or lines of text, using in place of words on small buttons, or anywhere you want to add some visual interest.

Alerts

Bootstrap alerts provide contextual feedback for users. The alerts are really simple and flexible, just wrap any text and an optional dismiss button in .alert and one of the four contextual classes (e.g., .alert-success).

Lists & Badges

Lists in bootstrap are very much like lists in plain html, the bonus is that they come with some very clean base styling. Also you can add badges with numbers or symbols.

I did find that I needed to add some css to the ol .list-group-item first to get the numbers to display (display: list-item) and then to get them inside the neat little rectangles (list-style: decimal inside).

Some other useful bootstrap components worth looking into include: dropdowns, breadcrumb navigation, pagination, the jumbotron header, progress bars, panels, and much more.

Modals

Modals are one of the javascript tools available in bootstrap. A modal is basically a pop-up window that blocks off background content.

You can apply pre-built sizing, formatting, and effect classes to your modals to personalize them to each site and instance.
